# 音频编解码芯片

音频编解码(codec)芯片选择的是**<font style="color:red;">ES8311</font>**, 该芯片以极高的性价比被广泛使用.

ES8311 是Everest Semiconduct(顺芯)公司推出的一款高性能、低功耗的音频编解码器（Audio CODEC）。它主要用于音频输入和输出的转换，广泛应用于各种音频处理设备，如智能手机、物联网设备、智能家居、穿戴设备等。

（1）ES8311 提供高质量的单声道音频解码和编码，支持多个音频采样率。

（2）它支持最高24-bit 音频精度, 8kHz-96kHz的采样率，能够提供高保真的音频输出，适合高端音频应用。

（3）ES8311 支持常见的音频接口，如 I2S（Inter-IC Sound）和 PCM（脉冲编码调制）接口，用于与微控制器（MCU）或其他音频处理器连接。

（4）还支持外部麦克风（MIC）和扬声器（Speaker）输入输出，能够方便地接入各种音频设备。

# LCD触摸显示屏

lcd显示屏选择的是 **<font style="color:red;">st7789</font>**.
ST7789 是一款由 **Solomon Systech（晶门科技）** 推出的高性能 TFT-LCD 控制器，广泛应用于中小尺寸（常见 1.3 英寸、1.54 英寸、2.0 英寸等）的彩色触摸显示屏模组中。它通过简化显示屏与主控芯片（如 STM32、ESP32、Arduino 等）的接口和控制逻辑，成为嵌入式设备、智能穿戴、手持终端等场景的主流选择。
一、核心功能与特性
ST7789 的核心价值在于 **“显示控制”**，即接收主控的图像数据并驱动 LCD 面板显示，其关键特性如下：

特性分类	具体说明
显示规格	- 支持最大分辨率：240×320 像素（QVGA 级别，适配主流中小尺寸面板）
- 色彩深度：16 位（65536 色，RGB565 格式，兼顾色彩与数据量）
- 支持屏幕旋转：0°/90°/180°/270°，适配横屏 / 竖屏场景
	接口类型	支持多种串行接口，降低硬件布线复杂度：
- SPI 接口（3 线 / 4 线，最常用，仅需 3-4 根信号线）
- I²C 接口（可选，适合对布线要求极高的场景）
	性能优化	- 内置显存（GRAM）：无需主控持续输出数据，可直接读取显存显示
- 低功耗设计：支持休眠模式，适合电池供电设备（如智能手环）
- 支持局部刷新：仅更新屏幕变化区域，降低数据传输量
	兼容性	适配多种 LCD 面板类型（IPS、TN 等），支持不同偏压设置，兼容性强
二、典型 “ST7789 触摸显示屏模组” 组成
单独的 ST7789 是显示控制器，市面上的 “ST7789 触摸显示屏” 通常是集成模组，包含 3 个核心部分：

ST7789 控制器芯片：模组的 “大脑”，负责解析主控指令、驱动 LCD 显示。
LCD 面板：物理显示部件，由 ST7789 驱动，提供图像输出（如 1.3 英寸 240×240 面板）。
触摸控制芯片：实现触摸功能（ST7789 本身不支持触摸），常见搭配芯片为 XPT2046（电阻触摸）或 FT6236/**GT911**（电容触摸），通过独立接口（如 SPI）与主控通信。
